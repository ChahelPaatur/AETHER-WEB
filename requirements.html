Get started

Requirements

  • Python 3.9+ on Linux, macOS, or Raspberry Pi OS.
  • Platform: Raspberry Pi 4 (primary target); also runs on a dev machine in --mode sim.
  • A robot kit — any supported HAT, or any kit that ships a Python library.
  • An Anthropic API key for the planner + vision (agent, server). sim and realworld run without it.

Core dependencies auto-install: numpy>=1.24, psutil>=5.9, requests>=2.32.
