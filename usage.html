Usage

Install & run

Pip install, drop in an API key, and run. You'll see auto-discovery scroll past, then type plain English at the objective> prompt.

Requirements

  • Python 3.9+ on Linux, macOS, or Raspberry Pi OS.
  • A robot kit — any supported kit, or any kit that ships a Python library.
  • An Anthropic API key — inference runs through the Claude API.
  • Optional: a Raspberry Pi 4 or 5. No GPU required.

Install

$ pip install aether-robotics

On Raspberry Pi OS Bookworm:

$ pip install aether-robotics --break-system-packages
$ export PATH="$HOME/.local/bin:$PATH"

Configure

$ export ANTHROPIC_API_KEY="your-key-here"

For persistence, add the line to ~/.bashrc.

Run

$ aether --mode agent

You'll see auto-discovery scroll past, then an objective> prompt. Type plain English:

objective> look up
objective> drive forward and stop 10 centimeters before the wall
›

Ctrl-C fires the full emergency-stop chain — every path ends with motors stopped and pins freed.

Quickstart · Raspberry Pi + PiCar-X

The full sequence from a fresh OS to running AETHER:

# enable I2C, SPI, Serial
$ sudo raspi-config

# system packages
$ sudo apt update && sudo apt install -y python3-pip git i2c-tools

# SunFounder libraries
$ git clone https://github.com/sunfounder/robot-hat.git
$ git clone https://github.com/sunfounder/picar-x.git
$ cd robot-hat && sudo python3 install.py
$ cd ../picar-x && sudo python3 setup.py install

# AETHER
$ pip install aether-robotics --break-system-packages
$ export PATH="$HOME/.local/bin:$PATH"
$ export ANTHROPIC_API_KEY="your-key"
$ aether --mode agent

You'll see [ADAPTER] Tier 1: sunfounder_picarx and [SKILLS] drive_until_obstacle in the boot log. Then go.

Python API

from aether.core.tool_registry import get_global_registry
from aether.skills.drive_until_obstacle import drive_until_obstacle

registry = get_global_registry()
registry.execute("move_forward", {"speed": 30, "duration": 2.0})

result = drive_until_obstacle(target_cm=10, max_speed=25, timeout_s=15)

HTTP API

$ aether --mode server  # starts on port 8000
$ curl -X POST http://localhost:8000/objective \
  -H "Content-Type: application/json" \
  -d '{"objective": "drive forward and stop 10 cm before the wall"}'

SymptomFix
aether: command not foundexport PATH="$HOME/.local/bin:$PATH"
Ultrasonic returns -1Check wiring: TRIG=D2, ECHO=D3, power, GND.
GPIO busyReboot, or sudo pkill -f python.
Sensor noise while drivingFixed in v3.9.5 (bad-read counter tolerates motor EMI).
